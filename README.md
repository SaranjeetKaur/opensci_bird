
# The Open Science Bird: Scrollytelling version

This repository provides a scrollytelling version of "The Open Science Bird" inspired and based on [Digtial Research Academy's](https://digital-research.academy/)'s [DRAdvent Calendar 2025 series](https://www.linkedin.com/posts/digital-research-academy_were-launching-our-dradvent-calendar-featuring-activity-7399800562126352385-UTAj). The content has been adapted to create an interactive experience that guides users through the principles and practices of open science.
